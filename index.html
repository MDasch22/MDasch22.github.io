Projects

Listed below you'll find some of the projects I am currently working on. Please feel free to explore each one of the sites by simply clicking on the picture.

Instashow | GitHub

Instashow is a solo full-stack pixel-perfect clone based on the popular social media platform, Instagram. This application provides a platform where users can interact with other users via posts/comments/likes/followers/ and the user update system. These features give the user a sense of personal customization in regards to what they are being displayed and how they choose to be displayed on the site.

Snacksy | GitHub

Snacksy was a group full-stack project clone of the popular e-commerce site Etsy. This application provides an avenue where snackers across the world can come together to share their love for all snacks. Users will have the ability to add snacks to our database as well as leave their thoughts and ideas on that particular snack which will be viewed by all other users.
